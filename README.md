# Executive Summary:
This analysis uses mock data to simulate the process of applicationg being submitted all the way to policies being issued for an insurance company. The funnel analysis aims to break down the application cycle and identify noticeable drop offs and make recommendations to help drive up conversion rate.

# Business Problem:
The insurance company has no problem generating leads or getting indiviudals to submit an application, however the conversion rate is around 3% and they'd like to improve this metric. The business suspects that there may be something going on during the application process, and need a data analyst to identify where in this process folks are dropping off.
